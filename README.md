# AI-Powered Meeting Assistant

An AI-powered application that automatically transcribes audio recordings and generates concise meeting summaries using state-of-the-art speech recognition and Large Language Models (LLMs).

## Overview

This project combines OpenAI Whisper, IBM WatsonX Llama 3, LangChain, and Gradio to create an end-to-end meeting assistant capable of:

- Converting speech to text
- Extracting key discussion points
- Generating meeting summaries
- Providing a user-friendly web interface

The application allows users to upload an audio file through a web browser, automatically transcribe its content, and receive a structured summary generated by a Large Language Model.

## Technologies Used

- OpenAI Whisper
- Hugging Face Transformers
- IBM WatsonX
- Meta Llama 3
- LangChain
- Gradio
- PyTorch
- FFmpeg

## Application Workflow

1. User uploads an audio file.
2. OpenAI Whisper transcribes the speech into text.
3. The transcript is passed to a LangChain prompt template.
4. IBM WatsonX Llama 3 analyzes the transcript.
5. Key discussion points and summaries are generated.
6. Results are displayed through the Gradio interface.

## Installation

Create and activate a virtual environment:

```bash
pip3 install virtualenv
virtualenv my_env
source my_env/bin/activate
```

Install dependencies:

```bash
pip install --force-reinstall "setuptools<70" \
transformers==4.36.0 \
torch==2.1.1 \
gradio==5.23.2 \
langchain==0.0.343 \
ibm_watson_machine_learning==1.0.339 \
openai-whisper
```

Install FFmpeg:

```bash
sudo apt update
sudo apt install ffmpeg -y
```

## Running the Application

```bash
python3 speech_analyzer.py
```

The application will be available through the Gradio web interface.
